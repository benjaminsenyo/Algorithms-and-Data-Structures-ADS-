\section*{Merge Intervals}

Pseudocode:
\begin{verbatim}
Input list of intervals
Sort intervals by start time
Create empty list merged
FOR each interval in intervals:
    IF merged is empty OR last interval in merged does not overlap:
        Add interval to merged
    ELSE:
        Merge current interval with last interval
RETURN merged
\end{verbatim}

Python implementation:
\begin{lstlisting}[language=Python]
def merge_intervals(intervals):
    if not intervals:
        return []
    intervals.sort(key=lambda x: x[0])
    merged = []
    for s, e in intervals:
        if not merged or merged[-1][1] < s:
            merged.append([s, e])
        else:
            merged[-1][1] = max(merged[-1][1], e)
    return merged

# Example:
# print(merge_intervals([[1,3],[2,6],[8,10],[15,18]]))
\end{lstlisting}

Complexity: O(n log n) time, O(n) additional space.
